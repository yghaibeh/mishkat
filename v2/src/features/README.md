# وحدات الميزات (Feature Modules) — المادة ٣/٢

كل ميزة **مجلد واحد** يضم: خادمها (`*.server.ts`) وخدماتها (`services/`) واختباراتها
(`__tests__/`) ومواصفتها (`SPEC.md`) ومستودعها في `db/repositories/`.
الاعتماد بين الوحدات **عبر واجهات معلنة فقط** — لا استيراد عابر للأحشاء.

الاتجاه الواحد للطبقات (المادة ٣/١)، تحرسه قاعدة الحدود في G2:

```
routes (عرض) ← server (تنسيق) ← services (منطق عمل) ← db (وصول بيانات)
```

ممنوع القفز (route يستورد من db) وممنوع الارتداد (service يستورد من route).

**المجلد فارغ عمداً**: نطاق T3-B هو السياج، وأول وحدة ميزة تولد **داخله**.
كل ميزة جديدة تبدأ بمواصفتها (G13) وتشير لطلب تغيير معتمد إن كانت جوهرية (G15).
