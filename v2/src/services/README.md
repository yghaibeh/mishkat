# طبقة الخدمات — منطق العمل

القواعد الحاكمة:
- **لا رقم تشغيلي صلب** (قب-٦): كل ثابت عمل يُقرأ من سجل الإعدادات عبر المُحلِّل المحقون.
  تحرسها **G14**.
- **لا استيراد لمكتبة قاعدة البيانات** (ADR-001 ع-١): المستودعات تُحقن. تحرسها **G17**.
- **لا فحص دور** (المادة ٤/٤): السؤال الوحيد هو `can()`. تحرسها **G6**.
- **الأخطاء قيم معلنة** (المادة ٣/٤): Result/Union لا استثناءات لأخطاء العمل.

المجلد فارغ من المنطق عمداً: **لا سطر ميزةٍ واحد قبل أن تكون البوابات خضراء على كود حقيقي**
(نطاق المهمة T3-B هو السياج وحده).
