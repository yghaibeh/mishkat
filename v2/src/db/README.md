# طبقة البيانات — حدود العزل (ADR-001 §٥)

> **هذا المجلد هو الموضع الوحيد المسموح فيه باستيراد مكتبة قاعدة البيانات.**
> تحرسه بوابة **G17**؛ وأي استيراد لـDrizzle/D1/SQL خام خارجه **يفشل الدمج**.

## الأحكام الستة الملزمة (ADR-001 §٥)

| # | الحكم | الحارس |
|---|---|---|
| ع-١ | لا SQL خام ولا استيراد Drizzle خارج `db/` — الميزات تنادي **مستودعاتٍ معلنة** فقط | G17 |
| ع-٢ | لا واجهة برمجة خاصة بـD1 في كود الميزات (`batch`, Sessions, bookmarks) | G17 |
| ع-٣ | **لهجة القاسم المشترك**: لا مزايا خاصة بمحرك | مراجعة + G17 |
| ع-٤ | يُكتب الكود كأنْ لا معاملات تفاعلية: عمليات idempotent بمفتاح طبيعي | مراجعة |
| ع-٥ | **كل جدول عمليات يحمل مفتاح توجيه** (مسار الوحدة) منذ أول هجرة | G10 + مراجعة المخطط |
| ع-٦ | الرولّ-أب **تجميعٌ مسبق حتماً**، لا مجموعٌ حيّ | مراجعة + مطابقة ليلية |

## البنية

```
db/
├── repositories/     مستودع لكل وحدة ميزة — العقد المعلن الذي تناديه الخدمات
└── (المخطط والهجرات — تُضاف في مهمة لاحقة بعد حسم مفتاح التوجيه)
```

## نقطة اللاعودة (ADR-001 §٦-١)

**مفتاح التوجيه (ع-٥) وموضع سجل التدقيق يُحسمان قبل أول هجرة تُدمج.**
بعدها يصير إصلاحهما هجرةَ بياناتٍ على ١٠٢ جدولاً لا قراراً على ورق.
أما ع-١…ع-٣ فنافذتها أوسع: **قبل وحدة الميزة العاشرة**.

> لذلك: هذا المجلد يحمل **العقود** ولا يحمل مخططاً بعد — المخطط قرارُ مهمةٍ تالية،
> وبناؤه هنا الآن كان سيسبق حسمَ نقطة اللاعودة.
