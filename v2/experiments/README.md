## المُقاس

| المرشّح | ما هو |
|---|---|
| **أ-١** | React 19 + تصييرٌ على الخادم + **ترطيب** (أرضيةُ الخيار أ: بلا موجّه ولا مكتبة استعلام) |
| **أ-٢** | ما سبق + **موجّه TanStack Router** (أقربُ ما بُني إلى شكل TanStack Start على العميل) |
| **ب** | **Hono** يقود الخادم + **جزيرةٌ واحدة** على العميل (طابور الأوفلاين) |

**الشاشة**: رئيسية أمير المسجد — `v2/src/features/home/screens/screens.ts` بقدرات الدور `amir`
من المصفوفة الذهبية. **لم تُكتب شاشةٌ جديدة**: `shared/tree.ts` يستدعي دالة الشاشة القائمة.

## لماذا القياسُ منصف

1. **شجرةُ عرضٍ واحدة**: المرشّحان يستهلكان `shared/tree.ts` نفسه.
2. **خريطةُ DOM واحدة**: `shared/mapping.ts` يقرّر الوسمَ والسماتِ والنصّ مرةً واحدة، ويُخرجه كلُّ
   مرشّحٍ بأدواته (React عناصرَ · سلسلةَ HTML). فالمقيسُ فرقُ **الإطار** لا اجتهادُ الكاتب.
3. **طابورُ الأوفلاين مشترك** (`shared/outbox.ts`): مطلبٌ معتمد (ت-٨/ب-٢٣) كلفتُه واحدةٌ للمرشّحَين،
   فلو حُسب لأحدهما وحده انحرف القياس.
4. **التحقّق**: المستندان متطابقان **بايتاً ببايت** بعد نزع وسم السكربت
   (`results.json ⟵ domIdentical.a1VsB = true`)، وعددُ عناصر DOM ٦٢ في الاثنين.

## إعادةُ التشغيل

```bash
cd v2/experiments/candidate-a && npm install --no-package-lock
cd ../candidate-b        && npm install --no-package-lock
cd ..
node measure.mjs        # يبني ويقيس البايتات ⟵ results/results.json
node field-model.mjs    # يحوّل البايتات زمناً بالسيناريوهات ⟵ results/field-model.json
node serve.mjs 4173     # ثم تُقاس أزمنةُ الترطيب في متصفّحٍ حقيقيّ (راجع results/browser-timings.json)
```

- **الإصدارات مثبّتةٌ حرفياً** في `package.json` لكل مرشّح (بلا `^`) — لا قفلَ مُلتزَم، فالتثبيت هو القفل.
- **ناتجُ البناء خارج الشجرة** (`$TMPDIR/mishkat-adr002r`، يُضبط بـ`MISHKAT_EXP_OUT`) كي لا يدخل
  ملفٌّ مبنيٌّ بوابةَ eslint (G2).
- الناتجُ **حتميّ**: لا تاريخَ ولا عشوائيةَ في المقيس؛ إعادةُ التشغيل تعطي البايتات نفسها.

## ما لا يقيسه هذا المجلد (يُقال ولا يُخمَّن)

- **لا CSS تخطيط**: ورقةُ الرموز اليوم متغيّراتٌ فقط (لا قواعدَ تخطيط) — فأرقامُ الحمولة **أرضيةٌ**
  ترتفع للمرشّحَين معاً بالمقدار نفسه عند كتابة تخطيط الشاشات.
- **لا TanStack Start كاملاً**: (أ-٢) موجّهٌ فقط، بلا زمنِ تشغيل دوالِّ الخادم ولا مكتبةِ استعلام —
  فالشكلُ الحقيقيّ للخيار (أ) **أثقلُ من (أ-٢) لا أخفّ**.
- **لا جهازَ ميدانٍ حقيقيّ ولا شبكةَ حقيقية**: معامل بطء الجهاز ٥×–٨× **افتراضٌ معلن**.
