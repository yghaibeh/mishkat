# مشروع المسجد المؤثر

منصة لإدارة شبكة مساجد (400 الآن، يُتوقع آلاف) — العمل الإداري والتعليمي والاجتماعي بنظام نقاط ومسابقة ومحاسبة مالية.

## بنية المستودع

| المجلد | المحتوى |
|---|---|
| [docs/](docs/) | المرجعان الأصليان: دليل العمل + مقترح المسابقة |
| [product/](product/) | وثائق المنتج (رؤية، أدوار، نموذج بيانات، PRD، قرارات، وحدتا المالية و«على بصيرة»، خطة البناء) + نسخ Word + نماذج تفاعلية |
| [app/](app/) | **التطبيق الكامل**: تصميم «مشكاة» (TanStack Start + React) + backend مدمج (Nitro + D1) — انظر [INTEGRATION.md](app/INTEGRATION.md). صفحة النظام التصميمي صفحة منفصلة `/design-system` لأصحاب المشروع فقط (admin، خارج تبويبات المستخدمين) |
| [platform/](platform/) | كود الـbackend المرجعي (Nuxt 3 + D1 + Drizzle) + 54 اختباراً + [دليل النشر](platform/DEPLOY.md) — منطقه منقول إلى التطبيق الكامل |

## الحالة

- ✅ التوثيق والقرارات: مكتملة ومعتمدة من الإدارة.
- ✅ النماذج التفاعلية: شاشة «سجل اليوم»، لوحة المحافظة، التقرير الشهري.
- ✅ **الـbackend مكتمل لكل الوحدات الموثّقة ومختبَر** (51 اختباراً): الأساس والصلاحيات، النقاط، المالية، «على بصيرة»، المسابقة، الحوكمة (دورة الإدارة)، المالية الداخلية للمسجد، الاجتماعات والقرارات، خطط اللجان، التحفيظ، الإشعارات، وتصلّب المصادقة — انظر [دليل المنصة](platform/README.md).
- ✅ **التصميم النهائي «مشكاة» مدموج بالـbackend** وعامل على workerd + D1: كل الصفحات تعرض بيانات D1 حقيقية بنفس التصميم تماماً.
- ✅ **شاشة دخول + مصادقة هوية كاملة** (cookie httpOnly + JWT، عزل نطاق حسب الدور) بلغة التصميم نفسها.
- ✅ **كل البيانات التجريبية في قاعدة البيانات** (`app/src/server/database/seed_demo.sql`) لا في الكود.
- ✅ Node v26 (الأحدث). مستخدمو البذرة: `admin` / `amir` (كلمة المرور `mishkat123`).
- ✅ **الـbackend مكتمل ومُختبَر** (54 اختباراً + e2e على workerd+D1): كل الوحدات + MFA + تصلّب المصادقة.
- ⏳ يتبقّى للإطلاق السحابي: `wrangler login` + `wrangler d1 create` + `database_id` + هجرات `--remote` + `wrangler deploy` (انظر دليل النشر). وربط هوية المستخدم/الصلاحيات بشاشة دخول (ميزة جديدة، الـbackend يدعمها). PDF/Cron يحتاجان إعداد Cloudflare.

## القرارات التقنية المعتمدة

Nuxt 3 · Cloudflare D1 (+ Drizzle) · كل الخدمات على Cloudflare · Web Push + بوت تيليغرام · PWA offline-first · عربي RTL + هجري.
