# عدة واجهات مشكاة

ثلاث شاشات تطبّق النظام، كلٌّ بالنمطين الفاتح (الافتراضي) والداكن:

- `amir-home.html` — شاشة ١: رئيسية أمير المسجد (جوّال 390px).
- `halaqa-log.html` — شاشة ٢: تسجيل يوم الحلقة للمحفّظ (جوّال 390px) — الحضور والتقييم تفاعليان.
- `admin-dashboard.html` — شاشة ٣: لوحة المدير العام (مكتب 1280px).
- `index.html` — عرض تفاعلي: مبدّل شاشات + مبدّل نمط.
- `screens.js` — قوالب الشاشات المشتركة + التفاعلات (وحدة ES).
- `kit.css` — أنماط الشاشات فوق رموز `styles.css`.

المحتوى النصي للشاشات ثابت من الموجز حرفياً؛ التبديل للنمط الداكن عبر `data-theme="dark"` على أي حاوية.
