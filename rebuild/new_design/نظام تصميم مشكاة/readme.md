# مشكاة — نظام التصميم

نظام تصميم عربي RTL لتطبيق «مِشكاة»: إدارة شبكة مساجد وحلقات قرآن (أمير المسجد، المحفّظ، المدير العام). النمط الفاتح افتراضي مع نسخة داكنة عبر `data-theme="dark"`. الأرقام عربية-هندية (١٢٣) دائماً. Mobile-First (390px) للأدوار الميدانية، وDesktop (1280px) للوحة الإدارة.

المصدر: موجز نصي من صاحب المشروع (لا Figma ولا كود سابق). المراسي اللونية المعطاة: ‎#1F6B45 / ‎#FBFAF7 / ‎#1B2420 / ‎#7A5104 / ‎#A32B22 — حُسّنت إلى سلالم كاملة.

## الروح
سكينة مسجدية بحداثة راقية: هدوء مصحف وترتيب تطبيق عالمي. الجمال بالفراغ والإيقاع والطبقات لا بالزخرفة. لا زخارف إسلامية مرسومة؛ الهوية تُبنى بالخضرة والدفء والمسافات.

## CONTENT FUNDAMENTALS
- اللغة: فصحى مبسّطة قريبة، مخاطبة مباشرة بضمير المتكلم/المخاطب («أين أنا من هدف الأسبوع؟»، «بقي عليك»).
- الأرقام: عربية-هندية حصراً (٤١ من ٧٠)، النسب بعلامة ٪ بعد الرقم (٨٩٪).
- التواريخ: هجرية أولاً («الثلاثاء ١٣ محرم»).
- نبرة الأخطاء: إنسانية مطمئنة، بلا لوم ولا مصطلحات تقنية.
- لا إيموجي إطلاقاً. الحالات بأسماء عربية قصيرة: معتمد / معلّق / متأخر.

## VISUAL FOUNDATIONS
- الألوان: أخضر ‎#1F6B45 هو الهوية والنجاح معاً؛ محايدات دافئة مائلة للأخضر؛ ذهبي هادئ ‎#8E7434 للتاريخ الهجري واللمسات فقط. خلفية ‎#FBFAF7 وسطوح بيضاء فوقها.
- الخط: IBM Plex Sans Arabic (400–700). سلّم 5 مقاسات: 30/22/17/15/13.
- المسافات: أساس 4px (‎--sp-1..9). الحواف سخية: 10/14/20/28. البطاقات: سطح أبيض + حد ‎--border + ‎--shadow-1؛ الظلال دافئة خفيفة، تشتد للطبقات العائمة (‎--shadow-3) وللأزرار اللاصقة (‎--shadow-sticky).
- الحركة: ‎cubic-bezier(.2,.8,.2,1)، ‏120/240ms. hover = ‎--surface-hover؛ الضغط = تعتيم طفيف. لا قفزات.
- التركيز: حلقة خضراء مزدوجة (‎--focus-ring).
- RTL كامل: البداية يمين؛ أيقونات التقدّم chevron-left.

## ICONOGRAPHY
لا أيقونات في المصدر؛ اعتمدنا Lucide من CDN (خط 2px، ينسجم مع الطابع الحديث الهادئ) عبر تقنية CSS mask لتلوينها بـ currentColor — **بديل مُعلَن يمكن استبداله بأيقونات خاصة لاحقاً**. لا شعار في المصدر: اسم «مشكاة» يُكتب بالخط العريض حيث يلزم الشعار.

## Intentional additions
كل المكوّنات من قائمة الموجز نفسها؛ أُضيف فقط `Icon` (غلاف لأيقونات Lucide) لضرورة تقنية.

## الفهرس
- `styles.css` — نقطة الدخول (تستورد `tokens/`: colors, typography, spacing, base, effects).
- `guidelines/` — بطاقات الأساسات: الألوان (أساسي/محايد/دلالي/داكن)، سلّم الخطوط، الأرقام، المسافات، الحواف والظلال.
- `components/` — المكوّنات (لكلٍّ `.jsx` + `.d.ts` + `.prompt.md` + بطاقة عرض):
  - `core/` — Icon، Avatar، StatusDot، Chip، Skeleton، format.js (أرقام عربية-هندية).
  - `actions/` — Button، StickyActionBar، SegmentedControl، AttendanceSwitch.
  - `navigation/` — AppBar، BottomNav، Sidebar، Tabs.
  - `data/` — StatCard، ProgressCard، ListItem، DataTable، Timeline.
  - `forms/` — TextField، SelectField، StepperField، SearchBar.
  - `feedback/` — EmptyState، ErrorState، Toast، BottomSheet.
- `ui_kits/mishkat/` — الشاشات الثلاث (فاتح + داكن) وعرض تفاعلي (`index.html`).
- `SKILL.md` — لاستعمال النظام من Claude Code.
