If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

Key rules: RTL Arabic only; Arabic-Indic numerals (١٢٣) everywhere; light theme default, dark via `data-theme="dark"`; IBM Plex Sans Arabic; green #1F6B45 is both identity and success; calm, ornament-free layouts built on whitespace and layering.
